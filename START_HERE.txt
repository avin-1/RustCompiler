✨ WHAT THIS COMPILER DOES
============================================================

Compiles a subset of Rust code through 6 phases:
   1. Lexical Analysis (tokenization)
   2. Syntax Analysis (parsing)
   3. Parse Tree (AST generation)
   4. Symbol Table (variable tracking)
   5. Intermediate Code (TAC generation)
   6. Optimization (constant folding)

Supports:
   ✓ 13 data types (i8-i128, u8-u128, f32, f64, bool)
   ✓ Variables (immutable & mutable)
   ✓ Operators (arithmetic, comparison)
   ✓ Control flow (if, while)
   ✓ Functions (with parameters & return types)
   ✓ Recursive functions
