============================================================
🆘 GETTING HELP
============================================================

If you're stuck:

1. Check the relevant documentation:
   - Installation issues → GETTING_STARTED.txt
   - Command syntax → QUICK_REFERENCE.txt
   - Process flow → EXECUTION_FLOWCHART.txt

2. Review example files:
   - test.rs (basic)
   - test_all_types.rs (types)
   - test_complex.rs (advanced)

3. Verify prerequisites:
   - Flex installed?
   - Bison installed?
   - GCC installed?
   - PATH configured?

4. Try standalone version first:
   - Simpler to build
   - No GTK3 required
   - Good for testing
